# Self-Host Weekly (17 April 2026)

> Indie computing, Linux 7.0, and a new Jellyfin client for...the Wii?

## Overview

![](https://cdn.jsdelivr.net/gh/selfhst/icons/svg/pikapods.svg)

I overheard a Mastodon user refer to self-hosting as *indie computing* this week and I can't stop thinking about it. Maybe it's the *coolness* factor, or just a welcome reprieve from the virtue signaling and crankiness that is becoming increasingly associated with self-hosting. Either way, I'm stealing it.

In actual news, there was a ton of headline-worthy activity this week. And because my love language is bullet points...

- **Cal.com** officially thinks open source is dead and [announced they're taking the platform closed source](https://cal.com/blog/cal-com-goes-closed-source-why?ref=selfh.st) for security-related reasons
- **Cloudflare** introduced the world to [Cloudflare Mesh](https://blog.cloudflare.com/mesh/?ref=selfh.st), a new private networking tool (mostly targeted at AI workflows) that may end up as a serious rival to Tailscale, NetBird, and similar platforms
- **Whoogle**, a privacy meta search engine, [dropped its final release](https://github.com/benbusby/whoogle-search/releases/tag/v1.2.4?ref=selfh.st) after years of working around Google's increasingly hostile scraping restrictions
- YouTuber **DB Tech** [published some great breakdowns](https://www.youtube.com/watch?v=S75WeLCOhZI&ref=selfh.st) outlining several recent open source supply chain attacks that are impacting self-hosted projects
- **Booklore** – yes, Booklore – [is back with a new repo and minor release](https://github.com/the-booklore/booklore/releases/tag/v2.2.2?ref=selfh.st) just a month after [completely dismantling any good will](https://selfh.st/weekly/2026-03-13/) the project had with the community
- **Allbirds**, the environmentally-friendly (and struggling) footwear manufacturer, [pulled a Kodak](https://www.forbes.com/sites/msolomon/2020/07/31/photography-to-pharmaceuticals-kodak-and-other-game-changing-corporate-pivots/?ref=selfh.st) and announced they're transitioning into [an AI company](https://www.wired.com/story/allbirds-is-pivoting-to-ai-compute-sure-why-not/?ref=selfh.st) (?)
- **Unraid** and **45HomeLab** [finally revealed their hardware collab](https://unraid.net/45homelabxunraid?ref=selfh.st), a pre-built server starting at – brace yourself – $2,999. (No need for sarcasm here, the jokes write themselves.)
- The developer of **SilverBullet**, a powerful note-taking tool, [launched a beta for SilverBullet+](https://community.silverbullet.md/t/announcing-silverbullet-beta-silverbullet-desktop-edition/4037?ref=selfh.st) – a desktop version of the software (currently macOS only) intended to help fund future development
- **Nextcloud** announced [Nextcloud Ethical AI Ratings](https://nextcloud.com/blog/nextcloud-ethical-ai-rating/?ref=selfh.st), a new tool to help users identify models that are open source, self-hostable, or trained using publicly available data
- **Raspberry Pi** **OS **has finally [disabled the default passwordless sudo in its v6.2 release](https://www.raspberrypi.com/news/a-security-update-for-raspberry-pi-os/?ref=selfh.st) *(something something OpenClaw...)*
- I experienced a bit of déjà vu as two different projects launched with the same name ([1](https://github.com/h4ckf0r0day/obscura?ref=selfh.st), [2](https://github.com/pauljoda/Obscura?ref=selfh.st)), while two other similarly named projects dropped v0.80.0 releases ([1](https://github.com/docmost/docmost/releases/tag/v0.80.0?ref=selfh.st), [2](https://github.com/kimdre/doco-cd/releases/tag/v0.80.0?ref=selfh.st)) within a day of each other
- The week's most absurd project launch name: [Boob O'Clock](https://github.com/liviro/boob-o-clock?ref=selfh.st) (infant sleep and feeding tracker)
- If you've been on the fence about making the jump away from Plex, wait no longer – someone finally released a [Jellyfin client for the Wii](https://github.com/fabienmillet/WiiFin?ref=selfh.st)

Happy **indie.comput**/ing!

## Content Spotlight

Meet **Transmute**, a privacy-focused file conversion platform. With **Transmute**, users can easily convert images, videos, audio, JSON, Excel, and more ([2,000+ supported conversions](https://transmute.sh/conversions/?ref=selfh.st)) into other formats via a minimal web interface. Features include client-side processing, single sign-on, unlimited file sizes, themes, and a REST API for automation.

**Transmute** can be [easily deployed via Docker](https://transmute.sh/docs/getting-started/?ref=selfh.st) and doesn't require any additional containers or databases to get up and running.

## Command Line Corner

Use **nl** to preface each line with a number when viewing the contents of a file from the command line. (This is particularly helpful when trying to reference lines in a large file.)

```bash
$ cat example.txt
  Self-
  Host
  Weekly
$ nl example.txt
  1 Self-
  2 Host
  3 Weekly
```
