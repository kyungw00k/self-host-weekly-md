## Weekly Highlights

This week, Linus Torvalds (the CEO of Linux) found himself [scolding the community for its backlash against AI](https://lore.kernel.org/all/CAHk-%3Dwi4zC%2BZe8e%2Bp3tMv8TtG_80KzsZ1syL9anBtmEh5Z40vg@mail.gmail.com/?ref=selfh.st) in response to development concerns over [Sashiko](https://github.com/sashiko-dev/sashiko?ref=selfh.st), an *agentic Linux kernel code review system*.

Naturally, Torvalds' remarks created a bit of a stir among both those in disagreement with his takes and those who find his methods of communication too crass. (TBD on how this will impact this week's milestone release of [Hannah Montana Linux v26](https://gitlab.com/DecaCagle/hannahmontanalinux26?ref=selfh.st).)

In other news, Windows also had a terrible, horrible, no good, very bad week as researchers [revealed a 0-day exploit](https://infosec.exchange/@wdormann/116925149776495861?ref=selfh.st) the same day Microsoft's Patch Tuesday release addressed a record-breaking number of security fixes.

Other activity and happenings you should be aware of:

- The team behind **Fluxer**, a promising [open source Discord alternative](https://github.com/fluxerapp/fluxer?ref=selfh.st), announced they're finalizing the project's API and self-hosting documentation (although maybe a bit too late given the release of [Chatto](https://github.com/chattocorp/chatto?ref=selfh.st), a newer alternative that [recently created some buzz](https://hmans.dev/blog/chatto-is-open-source?ref=selfh.st))
- [Immich](https://github.com/immich-app/immich/releases/tag/v3.0.3?ref=selfh.st) (photos) started pushing releases to [FUTO's official F-Droid repo](https://app.futo.org/fdroid/repo?ref=selfh.st) with their [v3.0.3 release](https://github.com/immich-app/immich/releases/tag/v3.0.3?ref=selfh.st) (for Android users who prefer to install apps outside of Google Play)
- **Plex** began [rolling out an updated interface](https://forums.plex.tv/t/changes-coming-to-plex-on-fire-tv/937614/3?ref=selfh.st) for its Roku and Fire TV Stick apps, and [users aren't happy about it](https://forums.plex.tv/tag/fire-tv/32?ref=selfh.st)
- **Dawarich** (location tracking) [added a poster studio in its latest release](https://github.com/Freika/dawarich/releases/tag/1.10.0?ref=selfh.st), which allows users to generate custom printable maps of their travels
- The latest **NocoDB** release, [2026.07.0](https://github.com/nocodb/nocodb/releases/tag/2026.07.0?ref=selfh.st), introduces new features available only to enterprise customers – a continued trend leaving the community increasingly frustrated ([1](https://old.reddit.com/r/NocoDB/comments/1uwxd4r/nocodbs_paywall_creep_finally_got_us_looking_for/?ref=selfh.st), [2](https://old.reddit.com/r/selfhosted/comments/1ux1ke8/nocodb_paywall_creep_looking_for_selfhosted_foss/?ref=selfh.st))
- Two relatively young and increasingly popular authentication platforms – [Pocket ID](https://github.com/pocket-id/pocket-id/releases/tag/v2.10.0?ref=selfh.st) and [Tinyauth](https://github.com/tinyauthapp/tinyauth/releases/tag/v5.1.0?ref=selfh.st) – announced they're officially OpenID Connect certified this week
- **Logseq** (wiki/collaboration) dropped its [initial beta release for v2.0](https://github.com/logseq/logseq/releases/tag/2.0.1?ref=selfh.st) while also announcing [the product is being split into two versions](https://logseq.io/p/e3YDyX5AYr?ref=selfh.st)

- Elon Musk [committed to publishing the source code](https://x.com/elonmusk/status/2077361679034118271?ref=selfh.st) behind **X** (formerly Twitter) after completing its current security vulnerability review
- **OpenAI** launched their first-ever physical device – the [Codex Micro](https://openai.com/supply/co-lab/work-louder/?ref=selfh.st), a $230 keypad controller for the platform's AI coding assistant

## Content Spotlight

Meet **SUB/WAVE**, a self-hosted and AI-powered personal internet radio station. With **SUB/WAVE**, users can broadcast a unified music stream (via Navidrome's Subsonic API) layered with weather, intros, time, and other commentary curated by an AI DJ. Features include user requests, swappable LLM providers, multiple DJ personas, multi-format broadcasts, native mobile apps, scheduled shows, pluggable skills, scrobbling, an MCP server, and a web-based console for administration.

SUB/WAVE can be deployed via [bare metal](https://github.com/perminder-klair/subwave?ref=selfh.st#quick-start-cli--recommended) or [Docker](https://github.com/perminder-klair/subwave?ref=selfh.st#quick-start-no-cli-raw-docker) and requires either a local LLM or a paid subscription to a hosted provider.

## Command Line Corner

Use **> file.txt** to flush the contents of a file from the command line (helpful for clearing old log files when troubleshooting via CLI):

```bash
$ cat example.txt
   Self-Host Weekly
$ > example.txt
$ cat example.txt
$
```
